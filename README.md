# Password Generator

Simple browser-based password generator.

## Description
Generates secure passwords using client-side JavaScript. Designed as a small demo and utility.

## Files
- index.html — main HTML file
- index.css — styles
- index.js — password generation logic

## Usage
1. Open `index.html` in your browser.
2. Configure options and generate a password.

For development, you can run a static server (optional):

```bash
# using Python 3
python -m http.server 8000
# then open http://localhost:8000
```
